
# Configuration

npm i -D @babel/register
From project root
mkdir test
cd test
touch mocha.opts
mkdir dbFunctions
cd dbFunctions
touch dbFunctions.test.js

**mocha.opts**
```js
--require babel-register
--watch
--recursive
```
